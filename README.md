# AI-Resume-Career-Advisor
AI Resume &amp; Career Advisor using Prompt Engineering, Retrieval-Augmented Generation (RAG), and Llama 3 for ATS evaluation, skill gap analysis, and personalized career guidance.
## Features
- ATS Resume Evaluation
- Skill Gap Analysis
- Resume Suggestions
- Career Guidance
- Personalized Learning Roadmap

## Technologies Used
- Python
- Groq API
- Llama 3
- PyPDF2
- python-docx
- Retrieval-Augmented Generation (RAG)
